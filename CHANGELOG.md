## [1.0.0] - 2025-03-09

### Added
- Initial release
- Automatic RTL detection and alignment for Hebrew text on any website
- Real-time handling for `input`, `textarea`, and `contenteditable` fields
- MutationObserver for dynamically loaded content (AI chat responses, etc.)
- Context-aware language detection via `<html lang>` and character frequency sampling
- Popup with enable/disable toggle and page rescan button
- Works on Claude, ChatGPT, SharePoint, and any other site
- Manifest V3 compliant
